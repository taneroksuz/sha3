# SHA3
Hardware and software implementation of SHA-3 algorithm. This algorithm is implemented in Verilog and C++.

## Hardware Implementation
This algorithm is implemented in finite state machine.

| Algorithm  | SHA3-224 | SHA3-256 | SHA3-384 | SHA3-512 |
|:-----------|:--------:|:--------:|:--------:|:--------:|
| LUT        | 4453     | 4420     | 4281     | 4142     |
| FF         | 1833     | 1865     | 1993     | 2121     |
| BRAM       | 0        | 0        | 0        | 0        |
